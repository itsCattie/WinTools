# Advanced Task Manager

Advanced Task Manager is a process and performance viewer inside WinTools.

It gives you a clearer look at what is running on your system than the built-in Windows task manager, with live performance graphs and the ability to group processes together.

## What it does

- Shows every running process with its name, PID, CPU usage, memory, and username.
- Lets you view processes in a tree (parent and child grouped together) or a flat table.
- Displays live performance graphs for CPU, memory, disk, and network over time.
- Lets you filter and search by process name.
- Lets you end a process or open its file location from a right-click menu.
- Supports grouping related processes so you can see their combined resource usage.

## How it feels to use

You open Advanced Task Manager and immediately see what is using the most CPU or memory. The graphs update in real time so you can watch a spike happen as you open something. Switching between tree view and table view lets you either trace why a process exists or sort purely by numbers.

Grouping is useful when you want to track a suite of related programs, for example a game plus its launcher and anti-cheat, as a single combined entry.

## Everyday flow

1. Open Advanced Task Manager from the WinTools launcher.
2. Sort by CPU or memory to find what is using the most resources.
3. Switch to tree view to see which processes spawned which children.
4. Check the performance graphs to see usage over the last few minutes.
5. Right-click a process to end it or navigate to its file.

## Quick note

Ending a process terminates it immediately without saving any work. Use it on background or stuck processes rather than ordinary apps.
