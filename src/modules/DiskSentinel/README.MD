# DiskSentinel

DiskSentinel is a disk space viewer inside WinTools.

It shows you exactly what is using space on your drives through a tree list and a colour-coded block map so you can find large files and folders at a glance.

## What it does

- Shows a card for each drive on your system with total, used, and free space.
- Scans any drive in the background while a progress bar keeps you updated.
- Displays results as a collapsible tree list with size, percentage, and item count per folder.
- Renders a treemap where each block's size matches how much space it takes up.
- Colours blocks by file type (images, video, documents, and so on) with a legend at the bottom.
- Lets you drill into any folder by clicking it in either the tree or the map.

## How it feels to use

You open DiskSentinel and see every drive laid out as a card. Clicking one kicks off a scan that runs in the background. Once it finishes the two panels fill in side by side. The tree on the left gives you the exact numbers and the map on the right makes the biggest offenders obvious straight away.

Scan results stay in memory for a few minutes so flipping between drives or navigating up and back feels instant.

## Everyday flow

1. Open DiskSentinel from the WinTools launcher.
2. Click a drive card to start a scan.
3. Click any folder in the tree list or block map to drill into it.
4. Use the **Up** button or the path bar to go back.
5. Press **Rescan** to refresh the results after moving or deleting files.

## Quick note

The first scan of a large drive can take a moment depending on how many files are on it. Subsequent navigation within the same scan is instant because results are already in memory.
